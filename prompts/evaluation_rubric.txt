
# 견고한 LLM 출력 품질 평가 시스템

## ️ 입력 형식 (필수)

평가를 위해 다음 XML 태그 구조를 **반드시** 사용해주세요:

<evaluation_request>
  <query>

  </query>

  <output_1>
    <title>[출력 1]</title>
    <content>

    </content>
  </output_1>
  
  <output_2>
    <title>[출력 2]</title>
    <content>

    </content>
  </output_2>

  <output_3>
    <title>[출력 3]</title>
    <content>
    
    </content>
  </output_3>
  
  <output_4>
    <title>[출력 4]</title>
    <content>
    
    </content>
  </output_4>
  
  
  <evaluation_context>
    <purpose>[평가 목적: 정보성/실행성/창의성/분석성/기타 중 선택]</purpose>
    <use_case>[실제 사용 상황이나 맥락 - 구체적으로 기술]</use_case>
    <priority_criteria>[중요하게 봐야 할 평가 기준 1-3개 명시]</priority_criteria>
    <comparison_type>[비교 방식: 2개_1대1_비교/다중_순위_평가/최고_선정 중 선택]</comparison_type>
    <target_audience>[대상 사용자: 일반인/전문가/학생/비즈니스 등]</target_audience>
    <special_notes>[기타 특별 고려사항이나 제약조건]</special_notes>
  </evaluation_context>
</evaluation_request>

###  입력 가이드라인

#### ✅ 필수 정보

- **query**: 평가 대상이 되는 원본 질문/요청
- **output_1, output_2**: 비교할 기본 두 출력 (최소 요구사항)
- **purpose**: 평가 목적 명시
- **comparison_type**: 비교 방식 명확화

####  선택 정보

- **output_3, output_4, output_5...**: 추가 출력 (다중 비교 시 순차 확장)
- **use_case, target_audience**: 맥락 정보
- **priority_criteria**: 중점 평가 기준
- **special_notes**: 특수 고려사항

##  평가 수행 프로세스

### 1단계: 입력 파싱 및 검증

- XML 태그 내용 정확히 추출 및 검증
- 비교 방식(1대1/다중/순위) 확인
- 평가 맥락 정보 파악 및 우선순위 설정
- 누락된 필수 정보 식별 및 요청

### 2단계: 정량적 평가 (100점 만점)

#### 완성도 지표 (30점)

- **정보 커버리지** (10점): 쿼리 요구사항 충족률
- **구조적 완성도** (10점): 논리적 흐름과 구성의 체계성
- **길이 적절성** (10점): 내용 대비 분량의 적정성

#### 정확성 지표 (30점)

- **사실 정확성** (10점): 검증 가능한 정보의 정확도
- **논리적 일관성** (10점): 내부 모순 없는 일관된 논리 전개
- **근거 제시도** (10점): 주장에 대한 적절한 근거와 출처 제공

#### 유용성 지표 (30점)

- **실행 가능성** (10점): 실제 적용/활용 가능한 정도
- **구체성** (10점): 추상적이지 않은 구체적이고 실용적인 정보
- **차별화 가치** (10점): 일반적이지 않은 고유한 인사이트나 접근법

#### 추가 지표 (10점)

- **사용자 친화성** (5점): 대상 독자가 이해하기 쉬운 정도
- **업데이트성** (5점): 최신 정보나 트렌드 반영도

### 3단계: 정성적 평가

#### A. 설명력 차원

- **명확성**: 이해하기 쉽고 모호하지 않은 설명
- **깊이**: 표면적이지 않은 심층적 분석과 통찰
- **맥락화**: 상황과 사용자에게 적절한 설명 방식

#### B. 정보 가치 차원

- **신선도**: 새롭거나 독창적인 관점과 접근
- **실용성**: 실제 문제 해결에 직접적으로 도움이 되는 정도
- **통찰력**: 숨겨진 패턴이나 연관성 발견 및 제시

### 4단계: 비교 분석 및 결론 도출
